Creative Features:
1. Sort Options - Sort search results by Title A-Z, Highest Rated, or Newest First
   using the filter button in the navigation bar. The current sort is indicated
   with a checkmark in the action sheet. Sort is applied locally without a new API call.

2. Search History - Last 5 searches are saved and shown as suggestions when
   tapping the search bar. Tap a suggestion to re-run that search. Use the
   "Clear" button in the header to remove all history. History persists across
   app launches via UserDefaults.

3. Context Menu Share - Long-press any movie poster to get a context menu with
   "Add to Favorites" (or "Remove from Favorites") and "Share" options without
   opening the detail view. This also satisfies the Extra Credit context menu
   requirement for favoriting directly from the search grid.
